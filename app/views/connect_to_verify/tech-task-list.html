Connect to GOV.UK Verify

Before starting these tasks, you should have planned your connection to Verify.

  1. 1.Development and testing

    Before you can connect to Integration, you must complete:

    • Send and receive messages from Verify
    • Build a matching service
    • Test in Integration

  2. 2.Connect to Integration

    Before you can connect to Production, you must complete:

    • Get certificates
    • Get access to Integration environment
    • Build and deploy in Integration

  3. 3.Connect to Production

    • Get certificates
    • Get access to Production environment
    • Build and deploy in Production

    Before going live:

    • Sign a memorandum of understanding
    • Pay for using GOV.UK Verify
