Connect to Verify

  1. 1.Set up connection

    Before you can connect to integration, you must complete:

    • Download and configure the Matching Service Adapter
    • Create Local Matching Service
    • Run compliance tool tests
    • Run matching service tool tests
    • Get certificates for Public Key Infrastructure

  2. 2.Connect to integration

    Before you can connect to production, you must complete:

    • Get access to integration environment
    • Configure Matching Service Adapter to connect to integration
    • Configure the service to connect with integration components
    • Run integration tests

  3. 3.Connect to production

    • Get access to production environment
    • Configure Matching Service Adapter to connect to production
    • Configure the service to connect with production components

    Before going live:

    • Sign a memorandum of understanding
    • Pay for using GOV.UK Verify
