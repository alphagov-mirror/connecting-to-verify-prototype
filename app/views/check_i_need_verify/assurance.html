Choose the right level of assurance

GOV.UK Verify offers 2 different levels of assurance.

Level 1

You can use level 1 if there aren’t any serious consequences to a user impersonating someone else when using your service.

Example The Driver and Vehicle Licensing Agency (DVLA) use level 1 in their view driving licence information service. This is because users can only get information from this service. They can’t use it to apply for anything more valuable or change their details.

Level 2

If there are serious consequences to a user impersonating someone else when using your service, you’ll need level 2.

You should also use level 2 if you need to able to take users to court when they give fraudulent information in your service.

Example HM Revenue and Customs (HMRC) use level 2 in their file Self Assessment tax return service. This is because a user could be fined if someone else pretends to be them and uses this service to give false information to HMRC.

Contact the GOV.UK Verify team if you think you might need a higher level of assurance for your service.
