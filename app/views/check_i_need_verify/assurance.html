Choose the right level of assurance

GOV.UK Verify offers 2 different levels of assurance (LOA). Which one you pick depends on how sure you need to be your users are who they say they are.

Levels of assurance

LOA1

You can use LOA 1 if there aren’t any serious consequences to a user impersonating someone else when using your service.

For example, the Driver and Vehicle Licensing Agency (DVLA) use LOA1 in their view driving licence information service. This is because users can only get information from this service - they can’t use it to apply for anything more valuable. It’s less likely a user would try to impersonate someone else to use this service.

It can only take 5 minutes for users to prove their identity if your service uses the LOA 1.

LOA2

If there are serious consequences to a user impersonating someone else when using your service, you’ll need the LOA 2.

For example, HM Revenue and Customs (HMRC) use LOA 2 in their file Self Assessment tax return service. This is because a user could be fined if someone else impersonates them and uses this service to give incorrect information to HMRC.

You should use the LOA 2 if you need to able to take users to court if they give fraudulent information in your service.

It will take up to 15 minutes for users to prove their identity if your service uses LOA 2.
