How to build a service with matching

Build a local matching service

The local matching service tries to match a user’s assured identity with a record in the government service’s data sources. This allows the government service to interact with the user. You need to build this yourself.
Download the local matching service

Set up Matching Service Adapter

The matching service adapter converts SAML, the language to communicate with the Verify hub into JSON for your service and vice versa. It also performs encryption, decryption, and signing of SAML messages.
Download the local matching service

Set up the Verify Service Provider

The Verify Service Provider helps create SAML requests and translate responses from SAML to JSON, which is more readable.
