Check if you need GOV.UK Verify Find out if GOV.UK Verify is right for your service

Use GOV.UK Verify if your service:

  • needs to legally confirm a user is who they say they are, for example if theyâ€™re signing a mortgage deed
  • shows a user personal information about themselves, such as their driving licence or passport details
  • pays money to a user
  • gives the user something valuable, like a Blue Badge or council house
  • replaces a form that needs to be physically signed by a user

When you shouldn't use GOV.UK Verify

Don't use GOV.UK Verify if your service:

  • receives payments from a user
  • doesn't keep any personal information about a user
  • needs a user to physically go somewhere (such as a court or Post Office) to prove their identity
  • needs to recognise a returning user but not check their identity - your service should let a user create an account instead
